# aerial-measure

Aerial roof measurement website designed to offer tools for measuring roofs using aerial imagery. The site will feature an intuitive interface for uploading images, selecting measurement parameters, and displaying results. The layout will be clean and professional, with a focus on usability and precision in the tools provided.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/aerial-measure.git
cd aerial-measure
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
